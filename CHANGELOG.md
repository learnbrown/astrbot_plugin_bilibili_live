# Changelog

本文档参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 的结构进行维护，并采用语义化版本号进行记录。

## [1.3.8] - 2026-05-27

### Changed

- 重写 `README.md`，统一项目说明、安装方式、指令说明、消息示例与兼容性说明的文档结构。
- 规范用户可见文本的表达风格，减少口语化和表情化措辞，统一为正式、清晰的提示语。
- 对支持 Markdown 的命令响应文本进行了结构化整理，便于在聊天界面中阅读。

### Fixed

- 修正首次订阅时的状态记录逻辑，避免主播在订阅当下已处于直播状态时，被后续轮询误判为“刚刚开播”。
- 整理轮询与持久化逻辑，减少不必要的重复写入，并补充订阅数据的兼容与清洗处理。
- 修正插件退出阶段对 HTTP 客户端的重复关闭问题，提升生命周期处理的稳定性。

### Refactored

- 抽取 B 站接口访问、主播昵称查询、状态文本映射与错误信息构造逻辑，降低重复代码并提升可维护性。
- 在关键逻辑处补充简洁注释，以说明状态记录、主动通知格式限制等实现原因。

## [1.3.7]

### Fixed

- 针对 AstrBot 主动消息推送暂不支持 Markdown 渲染的限制，将轮询开播通知调整为普通文本格式发送。
